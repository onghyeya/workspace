1. 객체지향 언어.

- 절차 지향적 방식
 > 코드의 실행 순서대로, 차례대로 작성
 ex) 차 :  엔진 > 타이어 >  문짝 > 바디


- 객체 지향적 방식( 절차 지향 + 객체 지향 )
 > 코드를 각각의 기능으로 분리해서 만들고
 > 분리된 내용을 조립해서 코드완성
  ex) 엔진 부서 :  엔진 // 절차 지향
      타이어 부서 :  타이어 // 절차 지향
      문짝 부서 :  문짝 // 절차 지향
      바디 부서 : 바디 // 절차 지향
      > 조립 = 객체 지향

객체(데이터와 기능으로 분리)
> 사람 
- 이름 ,나이, 주민등록증 = 데이터
