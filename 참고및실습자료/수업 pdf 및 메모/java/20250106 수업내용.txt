--------복습---------

< 클래스와 객체(인스턴스) >
- 실제 기능은 객체를 활용해서 작성!
- 객체 생성 : 클래스명 객체명 = new 생성자 호출;
-  클래스의 구성요소
: 멤버 변수, 메서드의 정의, 생성자
- 클래스의 구성요소인 멤버변수와 메서드의 사용방법
 > 객체명.멤버변수명, 객체.메서드 호출

< getter,setter >
- getter :  클래스 안에 정의 된 하나의 멤버변수의 값을 return하는 메서드
* 메서드명 :get멤버변수명
- setter : 클래스 안에 정의 된 하나의 멤버변수의 값을 변경하는 메서드
* 메서드명 : set멤버변수

< 생성자(constructor) >
- 문법 : public  클래스명(매개변수){}
- 역할 : 클래스에 선언된 멤버변수의 값을 초기화 할 용도
- 클래스 안에 생성자가 '하나도 없다면' default생성자를 생성
- default : public 클래스명(){}
- 메서드처럼 오버로딩이 가능
- 자동생성기능도 있음.

< this키워드 >
- 이 클래스에서 선언된 멤버변수를 가리킴

< 접근제한자 >
- 클래스, 멤버변수, 메서드 앞에 작성 가능
- 접근 범위를 지정 할때 사용
- 종류
> public :  같은 프로젝트
> default :  같은 패키지
> protected
> private : 선언한 클래스 내에서만
* 결론 
- 멤버변수는 무조건 private
- 메서드는 무조건 public
