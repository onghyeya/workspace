 -----getter/setter 자동완성(자주 씀)-----

<클래스 안>
마우스 우클릭 > generate 
> 멤버변수선택 후 getter/setter/getter and setter선택

<return>
> 결과 데이터를 컴퓨터가 나한테 돌려줌
내 입장 : 컴퓨터한테서 데이터를 받는다.

------------- 생성자 -------------

< 생성자 문법 > 
- public 클래스명(){}

< 목적 > 
- 멤버변수의 초기값 설정을 위해 생성자 사용
- 클래스 생성자가 하나도 없으면 기본생성자(default) 자동생성
- 생성자는 메서드 오버로딩 문법으로 여러개 생성가능
- 생성자를 하나라도 만들면, 기본생성자가 자동으로 만들어지지 않음.

------------접근 제한자------------

< 접근제한자 >
- 변수 , 메서드, 클래스의 사용 범위를 지정

< 종류 >
- public > default > protected > private
 
* public 선언된 변수와 메서드
: 같은 프로젝트라(project) 면 접근 가능
* default 선언된 변수와 메서드 
:  같은 패키지(package) 내에서 접근 가능, 키워드 작성 X
* private 선언된 변수와 메서드 
: 변수, 메서드를 선언한 클래스 안에서만 접근 가능
